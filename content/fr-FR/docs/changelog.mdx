# Dernières releases

Nous améliorons constamment O3. Cette page affiche la dernière release GitHub taguée pour une sélection de dépôts O3. C'est un tableau de bord de releases, pas un journal complet de tous les modules frontend.

Utilisez-la pour repérer les mouvements des packages O3 courants, puis vérifiez le fichier `spa-assemble-config.json` de votre distribution pour confirmer quels modules et quelles versions sont réellement inclus dans votre environnement. Chaque titre de dépôt et chaque lien "Voir toutes les releases" ouvre la page des releases GitHub du dépôt.
