# Journal des modifications

Nous améliorons constamment O3. Cette page suit les dernières releases du framework core et de quelques dépôts O3 qui couvrent la fiche patient, la gestion des patients, les formulaires, la saisie de données, la dispensation, les cohortes et le scaffolding de modules.

Chaque titre renvoie à la liste complète des releases GitHub, et le composant `LatestRelease` affiche les notes de la dernière release taguée. Vérifiez le fichier `spa-assemble-config.json` de votre distribution pour confirmer quels modules et quelles versions sont réellement inclus dans votre environnement.
