Latest releases

We’re constantly improving O3. This page shows the newest tagged GitHub release for selected O3 repositories. It is a release dashboard, not a full changelog for every frontend module.

Use it to spot when common O3 packages have moved, then check your distribution’s spa-assemble-config.json to confirm which modules and versions are actually included in your environment. Each repository heading and “View all releases” link opens the repository’s GitHub releases page.
